Holding Ball
===

Holding ball 프로젝트는 개인적인 학습을 목적으로 개발된 간단한 아케이드 게임입니다.
***

## 게임 규칙

### 득점 방식 
	볼이 돌아다니다가 날아오는 탄막에 겹치는 순간 길게 누르면 원 외곽선이 생성되며 점점커짐.
	이 외곽선이 다른 탄막에 닿는 순간 손을 떼면 외곽선 위의 탄막들의 갯수와 거리에 비례해 점수가 올라감.

### 게임 오버
	테마 색과 다른 색의 벽에 공이 닿았을 때 게임이 오버됨.

### 스킬
	벽에 닿을 때 마다 스킬이 초기화 됨.
	스킬은 드래그를 하여 사용하고 사용시 해당 방향으로 공을 1회 던질 수 있음.
